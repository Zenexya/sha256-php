Usage:
If you are going to use this on your local computer, use this:
http:/localhost?input=yourinputhere

Change the "yourinputhere" to text you'd like to encrypt.
The code will return an text hashed to sha256.

If you are going to use this on a website, use this:
yourwebsite.com/index.php?input=yourinputhere

As mentioned before, change the "yourinputhere" to text you'd like to encrypt.
The code will return an text hashed to sha256.
